# animeJSON
## Список всех аниме с сайта jut.su

> **Обновляю как получится** 

### **Список содержит:**
* Название
* Ссылку на аниме
* Ссылку на изображение
* Список сезонов, серий и фильмов
* Список "Жанры"
* Список "Темы"
* Список "Годы выпуска"
* Статус онгоинг
* Описание (не полное)
